# Security Policy

## Supported Versions

This project is currently an early prototype. Security reports are accepted for the latest code on the main branch.

## Reporting A Vulnerability

Please do not publish exploit details publicly before maintainers have time to respond.

When reporting, include:

- The affected browser and extension version.
- Steps to reproduce.
- Whether the issue requires a local HTML file or a remote page.
- Any relevant sample file with sensitive content removed.

## Privacy Notes

HTML-PPT Editor is designed to work locally:

- No account system.
- No cloud sync.
- No automatic upload of edited files or diagnostics.
- Drafts are stored locally in the browser.
