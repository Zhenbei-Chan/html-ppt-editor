# HTML-PPT Editor 开发者模式安装说明

## 1. 适用对象

本文用于内部测试人员在 Chrome 中安装 HTML-PPT Editor 开发者模式版本。

## 2. 安装方式

1. 打开 Chrome。
2. 进入 `chrome://extensions/`。
3. 打开右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择本项目根目录。

也可以先解压 `dist/html-ppt-editor-dev-v0.1.0.zip`，再选择解压后的目录。

## 3. 本地文件权限

如果需要编辑已经通过 `file://` 打开的本地 HTML 页面：

1. 在 `chrome://extensions/` 找到 HTML-PPT Editor。
2. 点击“详情”。
3. 开启“允许访问文件网址”。

如果只是通过插件入口“打开本地 HTML PPT 文件”，通常不需要额外设置。

## 4. 测试入口

安装后点击浏览器工具栏中的 HTML-PPT Editor 图标：

- “打开本地 HTML PPT 文件”：选择 `.html` 或 `.htm` 文件进入编辑器。
- “编辑当前页面”：对当前 Chrome 标签页注入编辑器。
- “体验多页示例”：打开内置多页 PPT 样本。
- “体验长页面示例”：打开内置长页面样本。

## 5. 验证清单

内部测试至少验证：

- 能打开 `.html` 文件。
- 能打开 `.htm` 文件。
- 能修改一段文字。
- 能新增文本框。
- 能移动和缩放文本框。
- 能撤销删除。
- 多页 PPT 能通过左侧缩略图切页、排序和删除。
- 长页面滚动时能看到画中画缩略导航，并能点击跳转。
- 能进入预览模式。
- 预览模式下页面原点击事件可用。
- 能导出新的 HTML。
- 能导出整份 PDF。
- 导出后的 HTML 能重新打开。

## 6. 已知边界

- 不支持图片替换作为首版核心能力。
- 不支持多选和组合。
- 不导出 PPTX。
- 不重新打包外部字体、图片、CSS 或 JS。
- PDF 是图片型 PDF，文字不可选中；部分高级 CSS、图标字体、远程资源可能存在渲染差异。
- 远程网页编辑只保存当前页面可编辑 DOM 修改，不保证完整复制动态网页。

## 7. 问题信息

遇到无法选择、无法编辑、导出异常等问题时，请记录：

- Chrome 版本和操作系统。
- 使用的是本地文件还是远程页面。
- 最小复现 HTML 样本。
- 编辑界面截图和导出结果截图。
- 相关控制台错误信息。
