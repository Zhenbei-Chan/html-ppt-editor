# HTML-PPT Editor Screen Spec

## 0. 总体界面原则

第一版界面只服务于“直接改 HTML PPT”。

设计判断优先级：

1. 画布内容优先。
2. 高频编辑操作可见。
3. 低频设置收纳到侧边属性面板。
4. 选中态、编辑态、预览态清晰区分。
5. 导出前必须预览确认。
6. 侧边工作区不悬浮覆盖页面。

---

## 1. 插件弹窗

参考图：`screenshots/01-plugin-popup.png`

### 1.1 界面目标

插件弹窗只做入口，不承载复杂编辑。

### 1.2 必备内容

主入口：

- 打开本地 HTML PPT 文件

次级入口：

- 编辑当前页面
- 打开多页示例
- 打开长页示例
- 问题信息
- 帮助

提示信息：

- 本地 `file://` 页面可能需要在插件设置中允许访问文件网址
- 所有编辑和导出默认在本地处理

### 1.3 视觉要求

- 暗色工具型界面。
- 主按钮使用蓝色。
- 次按钮弱化。
- 不放复杂说明。
- 不展示项目管理、文件列表或账号信息。

---

## 2. 编辑器选中态

参考图：`screenshots/02-editor-selected-state.png`

### 2.1 界面目标

用户进入编辑器后，可以选择页面元素并通过工具栏 / 属性面板修改。

### 2.2 布局结构

页面包含：

- 顶部应用栏
- 页面画布
- 侧边工作区
- 底部状态栏

侧边工作区包含：

- 窄列 icon 工具栏
- 属性面板

### 2.3 画布

画布显示原 HTML PPT 页面：

- 不额外套明显卡片边框。
- PPT 页面保留原比例。
- 长页面保留原滚动。
- 画布为主要视觉区域。

### 2.4 选中态

选中态用于单击元素后：

- 蓝色实线边框。
- 四角和边中显示缩放控制点。
- 只允许出现一个选中框。
- 属性面板显示当前元素位置、尺寸和文字属性。
- 选中页面导航按钮时，不触发页面跳转。

---

## 3. 编辑器文字编辑态

参考图：`screenshots/03-editor-text-editing-state.png`

### 3.1 界面目标

用户可以直接修改页面中的文字，并对选中部分文字做基础样式调整。

### 3.2 进入方式

- 双击元素。
- 或选中元素后点击工具栏“编辑文字”。

### 3.3 编辑态表现

- 橙色虚线边框。
- 显示文本光标。
- 可显示局部文字选中态。
- 不显示多个重复虚线框。
- 编辑态下方向键、空格、PageUp、PageDown 不触发原页面翻页。

### 3.4 样式规则

- 有局部文字选区时，加粗 / 颜色作用于选中范围。
- 没有局部文字选区时，加粗 / 颜色作用于整个当前元素。
- 字号、字体、对齐默认作用于整个元素。

---

## 4. 预览与导出确认态

参考图：`screenshots/04-preview-export-confirm.png`

### 4.1 界面目标

用户在导出前确认页面展示、翻页、动画和布局是否正常。

### 4.2 预览模式

预览模式隐藏：

- 工具栏
- 属性面板
- 选中框
- 缩放控制点
- 编辑器状态浮层

预览模式保留：

- 返回编辑
- 确认导出
- 预览模式标识

### 4.3 导出确认弹窗

弹窗包含：

- 文件名
- 当前修改数量
- 是否已进入预览确认
- 下载 HTML
- 返回编辑器

文案必须使用中文，不展示英文内部状态。

---

## 5. 状态栏

状态栏建议展示：

- 编辑模式 / 预览模式
- 草稿已保存 / 存在可恢复草稿
- 当前页
- 修改数量
- 缩放比例

文案示例：

- 编辑模式
- 预览模式
- 草稿已保存
- 存在可恢复草稿
- 第 3 页 / 共 12 页
- 修改 6 处

---

## 6. 响应式与长页处理

- 小屏时属性面板默认收起。
- 工具栏按钮过多时允许分组或滚动。
- 长页面滚动时，选中框位置需要跟随更新。
- 面板内部滚动不能抢占页面主滚动。
