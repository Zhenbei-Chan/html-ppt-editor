# HTML-PPT Editor UI Acceptance Checklist

## 1. 插件弹窗

- [ ] 弹窗只作为入口，不承载复杂编辑。
- [ ] 主按钮是“打开本地 HTML PPT 文件”。
- [ ] 次级入口包含“编辑当前页面”“打开多页示例”“打开长页示例”“问题信息”“帮助”。
- [ ] 显示 `file://` 权限提示。
- [ ] 显示“所有编辑和导出默认在本地处理”。
- [ ] 没有账号、云同步、文件管理后台类入口。

## 2. 编辑器布局

- [ ] 画布占据主要空间。
- [ ] 侧边工作区与画布并列，不悬浮覆盖。
- [ ] 工具栏和属性面板放在同一侧边工作区。
- [ ] 底部状态栏显示编辑状态、草稿状态、页码和修改数量。
- [ ] 页面内容没有被长期弹窗或面板遮挡。

## 3. 工具栏

- [ ] 工具栏采用窄列 icon 布局。
- [ ] 包含选择、移动、编辑文字、添加文本框。
- [ ] 包含字号、加粗、颜色、左对齐、居中、右对齐。
- [ ] 包含复制、删除、撤销、重做。
- [ ] 包含预览和导出。
- [ ] 当前模式按钮有选中态。
- [ ] 删除是弱危险色。
- [ ] 导出是主蓝色。
- [ ] icon button 有 tooltip。

## 4. 属性面板

- [ ] 顶部有“属性 / 当前页 / 已选择文本”。
- [ ] 包含页面、位置尺寸、文字、对齐、层级、列表、修改记录、问题信息。
- [ ] X、Y、宽、高是数字输入框。
- [ ] 字体为下拉选择。
- [ ] 字号为数字输入或步进器。
- [ ] 颜色有色块预览。
- [ ] 分组可折叠。
- [ ] 面板内部可滚动。
- [ ] 面板不覆盖画布和选中框控制点。

## 5. 选中态

- [ ] 单击元素进入选中态。
- [ ] 选中态为蓝色实线框。
- [ ] 四角和边中有控制点。
- [ ] 不出现多个重复选中框。
- [ ] 可拖动移动。
- [ ] 可拖动控制点缩放。
- [ ] 选中导航按钮时不触发跳转。

## 6. 编辑态

- [ ] 双击元素进入编辑态。
- [ ] 编辑态为橙色虚线框。
- [ ] 显示文本光标。
- [ ] 不出现多个重复虚线框。
- [ ] 可以直接输入文字。
- [ ] 可以选中部分文字单独加粗或改颜色。
- [ ] 编辑态下方向键、PageUp、PageDown、空格不触发原页面翻页。

## 7. 插入文本框

- [ ] 点击“添加文本框”进入插入模式。
- [ ] 光标样式变化。
- [ ] 画布显示插入位置提示。
- [ ] 点击位置创建文本框。
- [ ] 文本框不会默认插到页面角落。
- [ ] 创建后自动选中并进入编辑态。
- [ ] Esc 可以退出插入模式。

## 8. 预览

- [ ] 点击预览后隐藏工具栏、属性面板、选中框、控制点。
- [ ] 预览模式恢复原页面点击、翻页、动画和快捷键。
- [ ] 保留“返回编辑”入口。
- [ ] 保留“确认导出”入口。

## 9. 导出确认

- [ ] 导出前已进入预览确认。
- [ ] 弹窗显示文件名。
- [ ] 弹窗显示当前修改数量。
- [ ] 弹窗显示是否已进入预览确认。
- [ ] 主按钮为“下载 HTML”。
- [ ] 次按钮为“返回编辑器”。
- [ ] 不展示英文内部状态。
- [ ] 导出后 HTML 可以重新打开。
- [ ] 多页 PPT 导出后仍能翻页。

## 10. 不应出现的问题

- [ ] 不应做成网页管理后台。
- [ ] 不应出现长期遮挡画布的悬浮属性面板。
- [ ] 不应使用大面积装饰渐变。
- [ ] 不应把所有按钮都做成文字按钮。
- [ ] 不应默认扩展 AI、账号、云同步、图片替换、PPTX/PDF 导出。
